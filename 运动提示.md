# 运动提示参考

> 每条练习格式为 `[动作名, 参数, 提示]`。提示 = 执行时的注意力锚点，不是完整技术说明。

---

## 🏠 家里（徒手）

| 提示 | 对应动作 | 含义 |
|------|---------|------|
| 能完整说话 | 原地踏步 | 强度天花板：走到喘不出完整句子就说明太快了，减速。Zone 2 有氧标志。 |
| 慢下快上 | 深蹲 | 下蹲控制 2-3 秒，起来有爆发力。不追求次数，追求控制。 |
| 顶峰夹臀 | 臀桥 | 推到最高点时额外收紧臀肌一秒再放下。感受臀部发力而不是腰。 |
| 膝盖不超脚尖 | 弓步蹲 | 前腿膝盖垂直向下，不往前冲。身体重心在两腿之间。 |
| 核心收紧 | 跪姿俯卧撑 | 像有人要打你肚子一样绷住。从肩到膝一条直线，屁股不翘不塌。 |
| 不塌腰 | 平板支撑 | 腰不往下掉，骨盆微后倾。撑不住了就停，不硬撑坏姿势。 |
| 腰贴地 | 死虫式 | 全程腰椎压住地面不拱起。手和脚伸展时核心不松。做对 8 次比做错 20 次有用。 |
| 深呼吸 | 收尾拉伸 | 每个拉伸动作配合深长呼吸，不要憋气。 |

## 🏋️ 健身房

| 提示 | 对应动作 | 含义 |
|------|---------|------|
| 能完整说话 | 跑步机 | 同家里：走到喘不出完整句子就说明太快了。只走不跑。 |
| 保留 3 次余力 | 高位下拉 | 做完一组后还能再做 3 个。入门阶段不做到力竭，留空间给身体适应。 |
| 背挺直 | 低位划船 | 拉的时候背不弓。挺胸沉肩，先夹肩胛骨再用手臂拉。 |
| 不锁死肘 | 肩推 | 推到顶端肘关节不完全伸直，保持肌肉张力。锁死会把压力转移到关节。 |
| 不耸肩 | 侧平举 | 肩膀不往上缩，用三角肌中束发力。如果耸肩说明重量大了，减轻。 |
| 慢一点 | 腹部卷腹 | 不借惯性，不猛起猛落。每一次卷起和下落都控制节奏。 |
| 肩胛后收 | 坐姿划船 | 拉之前先把肩胛骨往中间夹，再用手臂拉。顺序对了背才练得到。 |
| 收尾 | 跑步机收尾 | 训练结束的低强度走动，帮助身体从训练状态平稳过渡。 |

## 🌳 户外

| 提示 | 对应动作 | 含义 |
|------|---------|------|
| 能完整说话 | 快走 | 同前：强度天花板是"能聊天"。不追求配速。 |
| 保持平衡 | 弓步走 | 每一步落地稳定后再走下一步。不稳就扶墙或减速。 |
| 找公园台阶 | 台阶上下 | 利用公共场所的台阶做训练。选高度合适的，不要太陡。 |
| 扶墙保持稳定 | 站立侧抬腿 | 一只手扶墙或柱子保持平衡，用髋外展发力抬腿，不靠身体甩。 |
| 背贴树 | 靠树静蹲 | 背完全贴住树干，大腿与地面平行或接近平行。膝盖不要超过脚尖太多。 |
| 找公园长椅 | 长椅臂屈伸 | 利用公园长椅做。手撑椅面，臀部悬空，弯曲手肘下降再推起。 |
| 不塌腰 | 草地平板支撑 | 同家里：核心收紧，身体一条直线。草地比硬地面更考验稳定。 |
| 深呼吸 | 慢走放松 | 训练结束后的冷却走。步子慢、呼吸深，让心率平缓回落。 |

## 全局提示

| 提示 | 适用场景 | 含义 |
|------|---------|------|
| 盖个小章 | 最低承诺日 | 最低承诺日唯一需要的心态：今天来过了，比昨天多一天。5 分钟也是胜利。 |
| 不断链 | 走路日记录 | "Don't break the chain"——每天做一点点，链子别断。哪怕只走 5 分钟写一笔。 |
| 入口 | 热身动作 | 训练的第一个动作，目的是把身体从休息状态过渡到运动状态。 |
| 收尾 | 结束动作 | 训练的最后一个动作，帮助身体平稳退出训练状态。 |

## 设计原则

这些提示来自 `references/minimum-friction.md` 的核心思路：

1. **降低执行门槛**：每个动作只需要记住一句话，不需要思考"怎么做才对"
2. **安全优先**：提示先保动作安全（不塌腰、不锁死、膝盖不超脚尖），再谈效果
3. **不惩罚失败**：最低承诺日的"盖个小章"是系统的核心——哪天实在没空，5 分钟也能不断链
4. **少即是多**：每个动作只有一条提示，多了反而记不住

需要新增动作或修改提示时，按以上原则增减。
